l  기능
Database로부터 얻어지는 데이터를 원하는 ISO 도면에 뿌려주는 기능을 합니다.

 

l  작업을 진행할 ISO Drawing이 있는 폴더를 사용자가 선택할 수 있습니다.(Option 탭)

 

 

l  쿼리 방법은 Edit Line Data 모듈부분을 참조하십시오.
쿼리한 후에 일치하는 ISO DRAWING을 FILE LIST에 표기합니다..

 : ISO DRAWING 파일이 존재할 경우.
 : ISO DRAWING 파일이 존재하지 않을 경우.

l  을 이용하여 Qualified Line 항목에 삭제/추가를 할 수 있습니다.

 

l  Qualified Line Form에 있는 Line Data에 대해서 Option에서 정의한 대로 ISO drawing에 Label/Text 출력 ( Reserved Data , Revision Data )

1.     옵션

- CONFIRM WITH USTATION : 작업 수행 후 UStation 실행
- LOG FILE SYNCHRONIZING : Log를 화면에 표시
- SAVE LOG FILE : Log를 파일로 저장
- REMOVE TEMP FILE : 기존에 Temp 폴더에 있던 파일들을 삭제
- QUALITY CONTROL CHECK : Q/C Check 기능 ON/OFF

2.      Log 파일은 Temp 폴더에 {생성된 시간}.LOG 파일로 생성됩니다.

 

l  Q/C Check
* 위의 옵션에서 QUALITY CONTROL CHECK 체크 박스가 체크되어 있을 경우에만 설정이 가능합니다.

1.     Error Option

1.     Error Reading Description 사항

2.     Piping Cutting Length Error

3.     B/M Not Found 사항

4.     Fail ISO Error

5.     RDB(*.pcd) File과 비교

6.     Support 개수 확인

7.     Hole Item 여부

8.     Material Code Error

9.     Text Modify 여부

2.     Warning Option

1.     Gasket / BOLT MTO 여부

2.     Spec. Break 여부

 

l   : Qualified 리스트에 들어있는 드로잉에 대하여 작업을 수행합니다.
à 작업이 완료된 파일은 Output Folder에 Output File Name.REV NO형식으로 저장됩니다.
( Output File Name 형식은 관리자가 설정한 값입니다)
